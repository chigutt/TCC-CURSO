CSS, O que é?

O CSS tem a função de separar o conteúdo do site de sua apresentação visual, alterando elementos como cor do texto, fonte e espaços entre os blocos, além de toda a parte estética da página.

Qual é o propósito do CSS?

Cascading Style Sheets (CSS) é a linguagem de marcação responsável pela aparência do seu site. Ele controla as cores, fontes e layout dos elementos do seu site. Essa linguagem de estilo também permite adicionar efeitos ou animações ao seu site.

As principais tags:

    Color, faz com que mude a cor da letra.
    Backgroud-color, faz com que o fundo mude de cor.
    Padding, Usado para definir o espaçamento ao redor de um elemento.
    margin, define a margem nos quatro lados do elemento.
