HTML, oque é?

O HTML (HyperText Markup Language) é a linguagem de programação mais básico da web. Ele define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML são frequentemente usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Qual a importância do HTML?

HTML é a base da web. Portanto, quem deseja se tornar um programador deve dominar esta linguagem e dominar seus códigos de funcionamento. Qualquer pessoa familiarizada com a Internet e a tecnologia já ouviu ou leu o termo HTML.

As principais tags:

    <.!DOCTYPE>, que diz ao navegador qual a versão que está o HTML.
    <.body>, Indica ao navegador o corpo do documento HTML, resumindo a parte principal do site.
    <.div>, Indica a tag que funciona como um contêiner e define um bloco inteiro de conteúdo.
    <.title>, Define o titulo do documento, o nome que fica no topo da janela.
    <.class>, cria uma classe, fazendo ser possível ser modificado no CSS.
